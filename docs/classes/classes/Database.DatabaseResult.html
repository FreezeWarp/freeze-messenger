getAsArray()

getAsArray(boolean|string  $index = true, boolean  $group = false) : array

Get database resultset as a single associative array or multiple associative arrays. An empty array will be returned if an error occurs.

Parameters

boolean|string $index

When this is a string, it corresponds to a column whose value will be used to index the returned arrays. When it is true, the returned arrays will be indexed automatically, starting at 0. When it is false, only one array will be returned.
boolean $group

If true, results will be grouped by the index, e.g. if you group on "entries", [1 => [["id" => 1, "entries" => 1], ["id" => 2, "entries" => 1]]] will be returned. If $group were false, [1 => ["id" => 1, "entries" => 1]] would be returned instead.

Returns

array

getAsTemplate(string  $format) : mixed

Get the database object as a string, using the specified format/template. Each result will be passed to this template and stored in a string, which will be appended to the entire result.
