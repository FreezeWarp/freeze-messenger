FileCache - A simple file based cache

Class to implement a file based cache. This is useful for caching large objects such as API/Curl responses or HTML results that aren't well suited to storing in small memory caches or are infrequently accessed but are still expensive to generate.

For security reasons, it's strongly recommended you set your cache directory to be outside of your web root and on a drive independent of your operating system.

Uses JSON to serialize the data object.

Sample usage:

$cache = new FileCache('/var/www/cache/'); $data = $cache->get('sampledata'); if(!$data){ $data = array('a'=>1,'b'=>2,'c'=>3); $cache->set('sampledata', $data, 3600); } print $data['a'];

Changes by Joseph T. Parsons:

  • FIM syntax style
  • exists function
  • add function
  • dump function
  • serialize instead of json
